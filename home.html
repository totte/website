The premiere KDE experience.

Chakra is a GNU/Linux distribution with an emphasis on KDE and Qt technologies that focuses on simplicity from a technical standpoint and free software.

Stable core, bleeding edge applications.

With our half-rolling release model we provide a thoroughly tested core layer of software - such as the Linux kernel, GNU coreutils and common libraries - while the software in the applications layer is updated more frequently.

Independent.

Derived from Arch Linux(tm) but not based or built upon any other distribution. Don't phrase this in a way that makes it seem as if we are mooching off their brand, but make the origin (if necessary) and current independence clear. Arch Linux(tm) is still to be properly acknowledged in the credits/acknowledgements on another page. Make it clear that what we do is built from scratch - we don't copy other packages that have already been built into other repositories.

Brought to you by a diverse and skilled community.

The Chakra contributors are a handful of avid hackers from all around the world whom have come together, sharing a fondness for the elegance of the Plasma Workspaces and a desire to create and share useful software for everyone to enjoy.
